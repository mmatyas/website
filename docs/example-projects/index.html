Who's Using Neon

Here is a list of projects using Neon

#Node Libraries

  • swc: Super-fast Babel alternative written in Rust
  • disk-utility: Disk utility methods for Node using Neon
  • node-fuzzy-phrase: Fast fuzzy phrase matching for Node
  • snips-nlu-js: Quick and dirty Node bindings for snips-nlu-rs
  • reconfix: (Re)Configuration toolkit
  • libsodium-neon: Node bindings to libsodium
  • ledb: Lightweight embedded database
  • node-crypto: rust-crypto bindings for Node
  • os-type: Bindings to the os_type rust library
  • node-webrender: bindings to webrender

And many more!

#Electron Apps

  • finda: Finda finds things — in under 16 milliseconds.

#Rust Libraries

  • neon-serde: Easily convert between JsHandle and serde structs

Want to add a project to this list? Submit a PR!
